# 第四章 修饰符

| 修饰符 | 含义        | 说明                              |
| ------ | ----------- | --------------------------------- |
| `i`    | ignore      | 不区分大小写                      |
| `g`    | global      | 全局匹配                          |
| `m`    | mutil line  | 多行模式，使`^` `$`在每一行都生效 |
| `s`    | single line | 单行模式，使.包含换行符           |
